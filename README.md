js-structure
==============

 use javascript to complete common data strutures(通过javascript实现常见的数据结构)


## API

### Stack(栈)

``` javascript

Method

>push(): 入栈

>pop(): 出栈

>getTop(): 获取栈顶元素

>getLength(): 获取栈的长度

>clear(): 清空栈

>copy(): 复制栈

>revert(): 回文操作

>isEmpty(): 判断空栈

Property

>dataSet: 数据存储对象

>top: 栈顶指针

```


-----------------


### SingleChain(单向链表)

``` javascript

Method

>find(): 查找节点

>findPrev(): 查找前一个节点

>insertBefore(): 插入到某个节点之前

>insertBefore(): 插入到某个节点之后

>remove(): 移出某个节点

Property

>dataSet: 数据存储对象

>head: 头结点

```
