js-structure
==============

 use javascript to complete common data strutures(通过javascript实现常见的数据结构)


## API

### Stack(栈)

``` javascript

Method

>push(): 入栈

>pop(): 出栈

>getTop(): 获取栈顶元素

>getLength(): 获取栈的长度

>clear(): 清空栈

>copy(): 复制栈

>revert(): 回文操作

>isEmpty(): 判断空栈

Property

>dataSet: 数据存储对象

>top: 栈顶指针

```
